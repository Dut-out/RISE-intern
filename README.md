# To-Do list Web page
To-Do list is an application which provide user to manage there daily task by creating a list in which they can mark there task.

## Built using
### Frontend
- BootStrap
### Backend
- Php and MySql

## Features
- CURD Operation
- Mark tasks
- Task filtering
- Responsive Design
